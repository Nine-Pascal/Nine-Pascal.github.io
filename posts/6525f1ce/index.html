Git安装与使用

macOS 下安装 Git

首先检查电脑里面是否有安装过Git，按下 command + space，搜索terminal

在 macOS 上通过 homebrew 安装 git

在安装过程中会要求输入密码（Password），输入密码过程不可见，密码输入完整后回车即可继续

因为安装 homebrew 的前提是电脑里安装了 git，所以在检测到电脑里面没有 git 后会弹窗要求安装，统一安装后等待即可。

通过 Xcode 安装

直接从 AppStore 安装 Xcode，Xcode 集成了 Git，不过默认没有安装，你需要运行 Xcode，选择菜单 “Xcode”->“Preferences”，在弹出窗口中找到 “Downloads”，选择 “Command Line Tools”，点 “Install”就可以完成安装了。
